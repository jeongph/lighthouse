## 작업 내용

jeongph-workspace의 역할(Claude Code 최상위 지침 + 워크스페이스 루트)과 커밋 히스토리를 lighthouse 레포로 통합하고, lighthouse를 새 워크스페이스 루트로 전환했다.

## 수행 단계

1. **백업**: jeongph-workspace 전체를 tar.gz로 백업
2. **guide/ → handy 이관**: lighthouse/guide/ 3개 파일(Claude_Code.md, tmux.md, gh.md)을 handy/guide/로 이동
3. **불필요 디렉토리 제거**: lighthouse에서 guide/, development/ 삭제
4. **히스토리 merge**: `git merge --allow-unrelated-histories`로 jeongph-workspace 커밋 히스토리를 lighthouse에 통합
5. **.gitignore 재작성**: 워크스페이스 루트 + conventions/, docs/history/ 허용 구조로 변경
6. **git-commit-convention.md 수정**: other, init 타입 제거, chore 타입 추가
7. **CLAUDE.md 재작성**: lighthouse 맥락에 맞게 전면 재작성 (제목, 개요, Git 정책, 레포 목록 등)
8. **README.md 수정**: 워크스페이스 구조 섹션 추가
9. **물리적 이동**: lighthouse를 /home/jeonguk/lighthouse로 이동, handy와 helpful-feeds를 그 안으로 배치
10. **push**: lighthouse, handy 원격 push
